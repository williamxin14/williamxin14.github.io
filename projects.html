MY PROJECTS

Classwork

United States Historical Boundaries App

Created an interactive web map documenting US Historical State Boundaries. The web app showcases all historical boundary changes ranging from 1783 to 1959.

Emergency Relief Zones

Created hypothetical Relief Zone Map using Proximity and Overlay Analysis in ArcGIS Pro for Emergency/Disaster Management. Analysis Tools such as Buffer lines and Intersect operations were used.

Data Classification Tool Script

Created an ArcGIS toolscript that automatically joins csv tables, defines interval classifications, and labels classes.

Digitalizing Historical Maps

Digitalized a paper scanned 1920s Sanborn map of Charlottesville, Virgnia. The scope of the project focused on digitalizing a few blocks within the city center.

Site Selection Analysis

Performed Site Selection Analysis to plan a biological reserve system within Centre County. Created a set of criteria for suitable land use of Conservation and used ArcGIS to idenify areas

Personal

Illinois 2020 Census App

Created an interactive choropleth map web map that allows the user to explore racial demographics of each Census Tract and each County in Illinois for the latest Decennial Census.

Geography of Illinois

Created an Layout of Illinois Geography using ArcGIS pro. This layout contains a 30 meter DEM map, and HUC level Sub Basin Watershed Map with the Basins outlined.

3D Population Density Map

Created a 3D Census Tract Map of Illinois with the height determined by population density. Map is created using ArcGIS Pro Scenes. Data is from the 2020 Census
