Data Classification Tool Script

Introduction

Final Project for GIS Programming and Software Development

Created a Python Tool Script that uses a defined interval classification method ,automatically labels the classes, and join a CSV data file. This script tool is flexible enough to be used for multiple datasets

Parameters

The first four parameters is used for joining a csv table to the targeted vector polygon layer. After setting an interval the tool script will automatically determine the largest and smallest class break and covers the range of negative values. The tool script also allows the user to round up numbers for labels and can toggle betwteen displaying the legend in numbers and percentages.
